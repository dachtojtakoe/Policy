# Policy
Политика конфиденциальности

Настоящая Политика конфиденциальности регулирует порядок использования и хранения данных для мобильного приложения MeteoMobile (далее – «Приложение»).

1. Сбор и использование данных

Приложение является клиентом для отображения метеорологической информации. Приложение не собирает, не обрабатывает и не хранит какие-либо персональные данные пользователей.

Для своей основной функции Приложение подключается к внешней базе данных для получения актуальных метеорологических данных. В процессе этого соединения технически могут передаваться обезличенные данные, необходимые для установки соединения (например, IP-адрес). Эти данные не используются для идентификации пользователя.

2. Данные третьих сторон

Приложение отображает данные, полученные из внешних источников (баз данных). Мы не несем ответственности за содержание, точность и политику конфиденциальности этих внешних источников.

3. Изменения в Политике конфиденциальности

Мы оставляем за собой право вносить изменения в настоящую Политику конфиденциальности. Новая редакция Политики вступает в силу с момента ее публикации внутри Приложения или на странице его описания в магазине приложений.
